---
title: "卸载"
---

注意：卸载将从机器中删除 KubeSphere，该卸载操作不可逆，也不会备份任何数据，请谨慎操作。

**1.** 在 "/scripts" 目录下执行 `menu.sh` 脚本：

```bash
$ ./menu.sh
```

**2.** 输入数字 `4` 选择 Uninstall 卸载软件：

```bash
################################################
         KubeSphere Installer Menu
################################################
*   1) All-in-one
*   2) Multi-node
*   3) Add-node(s)
*   4) Uninstall
*   5) Quit
################################################
https://kubesphere.io/               2018-11-16
################################################
Please input an option: 4
```
